## smui 使用说明

### 安装

***npm导入项目依赖***

```sh
cd myproject

npm install smui -s -d

```

***引入模块***

```js

// 引入所有模块

import * as smui from 'smui'

// 引入部分模块, 仍然会加载所有组件

import {Table, Calendar} from 'smui'

// 按需导入文件

import Table from 'smui/Table'
import Pager from 'smui/Pager'

```
