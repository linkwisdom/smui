## 使用安装

* NPM安装

npm i smui \# for vue1.x

npm i smui-next \# for vue2.x

npm i smui-souce \# souce code



* CDN安装

[http://image.uc.cn/s/uae/g/2p/smui/product/smui-0.0.11.js](http://image.uc.cn/s/uae/g/2p/smui/product/smui-0.0.11.js)
